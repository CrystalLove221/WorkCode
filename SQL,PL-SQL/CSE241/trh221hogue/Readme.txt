This program connects to Lehigh's messed up university database, searches for instructors matching some user-provided substring, and prints the teaching record of some instructor, given some user-provided instructor ID.

Legal jargon: This program is open-source. The manifest is left for the user to make changes and additions to the code to suit their database needs.

ojdbc8.jar: JDBC package to support communication with an SQL database.
TeachingRecord.jar: main program
